Project Description

Each of these seven games is useful for Web Developers learning the fundamentals of vanilla JavaScript. And they can be accessed by clicking on the numbers at the top of the widget found above.

This personal project (developed during October of 2020) demonstrates how a healthy amount of educational interactivity can be framed inside of a responsive widget. Which is intended to be placed on the sidebar and/or homepage of a website for improved engagement with users.
